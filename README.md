## ✅ Features Implemented

- ✅ User Registration & Login (with password encryption)
- ✅ User Profiles with bio
- ✅ Create, View, Delete Posts
- ✅ Like/Unlike Posts
- ✅ Comment System
- ✅ Follow/Unfollow Users
- ✅ Real-time Feed
- ✅ Responsive Design

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript (Vanilla)

**Backend:**
- Node.js
- Express.js
- bcrypt (Password hashing)

**Database:**
- JSON File-based storage
- Tables: users, posts, comments, followers, likes
